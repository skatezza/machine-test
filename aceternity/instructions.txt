# Agency boilerplate project

## Project overview
This project is focused on developing a reusable boilerplate for the Agency landing page template.

## Project goals
Our goals for this project include:
- Centralizing all the copywriting of the website + image assets referrencing in a single copy-config.ts file, which you'll help create 
- Replacing all the copywriting + image assets in the website with the ones in the copy-config.ts file

## instructions
- You will create the copy-config.ts file in the src/app/copy directory
- You will use the copy-config.ts file to update the landing page copy + image assets referrencing in the website

## Code quality and standards
- We aim to keep the code minimal across all sections
- Emphasis on maintaining highly reusable sections
